# Objective
This  focuses on manual testing to evaluate a website’s design and functionality.

# Task
Manually test the homepage of the website Lethal Demo(https://lethaldemo.com/digct/) to ensure it matches the provided JPEG(https://themenio.com/assignment.html) design.

## Testing Focus
- Verify **pixel-perfect alignment, spacing, and colors** against the provided design.  
- Test all **links, buttons, and the contact form** for proper functionality.  
- Ensure **responsiveness** across different devices (desktop, tablet, mobile).  
- Test **cross-browser compatibility** (Chrome, Firefox, Edge).  
- Document any design or functional issues observed during testing.  

---

## Issue Reporting
- All issues are documented with **screenshots** in **Google Docs/Sheets**.  
- Each issue includes **reproduction steps** to assist developers in fixing bugs.  
- Issues are categorized by **design, functionality, responsiveness, or compatibility**.  

---

## Tools & Environment
- **Browsers:** Google Chrome, Firefox, Edge  
- **Devices:** Desktop, Tablet, Mobile  
- **Documentation:** Google Docs / Google Sheets  

---

## Achievements / Highlights
- Validated the homepage against the reference JPEG design for **pixel-perfect accuracy**.  
- Ensured **all interactive elements** (buttons, links, forms) function correctly.  
- Confirmed website **responsive behavior** across multiple devices and browsers.  
- Reported all findings with clear **reproduction steps and screenshots** for developers.
